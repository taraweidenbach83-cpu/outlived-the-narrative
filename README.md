# Outlived the Narrative

Interactive workbooks and guides for families navigating rare disease diagnosis and medically complex caregiving.

## 🎯 Mission

**Outlived the Narrative** provides cinematic, trauma-informed, emotionally intelligent resources for families facing:
- AGS (Aicardi-Goutières Syndrome) diagnosis
- Medically complex caregiving
- Rare disease navigation
- Medical trauma recovery

## 🎨 Experience

- **Cinematic & Emotional**: Dark mode with purple/teal accents, calming animations
- **Trauma-Informed**: Gentle pacing, permission to feel, no judgment
- **Premium & Professional**: Modern design, accessible, scalable
- **Interactive**: Editable journals, checklists, exercises, progress tracking
- **Portable**: Downloadable PDFs, printable worksheets, mobile-friendly

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## 📁 Project Structure

```
src/
├── app/                 # Next.js app directory
├── components/          # Reusable React components
├── content/            # Workbook content data
├── hooks/              # Custom React hooks
├── lib/                # Utilities & helpers
├── styles/             # Global styles
└── utils/              # Helper functions
```

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **State Management**: Zustand
- **PDF Export**: jsPDF + html2canvas
- **Icons**: Lucide React
